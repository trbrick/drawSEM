# Visual Web Tool

Minimal Vite + React + TypeScript project with a small SVG-based canvas tool (rectangle add & drag).

Getting started

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build

```bash
npm run build
```

4. Preview build

```bash
npm run preview
```

Notes
- Recommended VS Code extensions: ESLint, Prettier, Tailwind CSS IntelliSense.
- Project uses Tailwind for styling; configuration is in `tailwind.config.cjs`.
